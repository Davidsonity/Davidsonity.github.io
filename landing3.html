Heart Failure Classifier

The objective of this project is to build machine learning models to predict heart failure of a patient. This dataset contains the medical records of 299 patients who had heart failure, collected during their follow-up period, where each patient profile has 13 clinical features.

Nature Scene Classifier

The Intel Image Classification project involves the classification of images into different categories using deep learning techniques. The project focuses on a dataset provided by Intel, which consists of images of various natural scenes such as buildings, forests, glaciers, mountains, seas, and streets. The main objective of the project is to develop a deep learning model capable of accurately classifying images into these predefined categories. By training the model on a large dataset of labeled images, the project aims to achieve high accuracy in predicting the correct category for new, unseen images. The Intel Image Classification project has applications in image recognition, computer vision, and automated classification tasks.

Dog Breed Classifier

The objective of this project is to build powerful neural network that can classify dogs according to their breeds. The dataset comprises 120 breeds of dogs. For this project, the top 10 most populated dog breed in the dataset are used to train the model.
