Football Fixtures Scraper

The Football Fixtures Scraper is a web application deployed using Streamlit. It allows users to fetch and analyze football match fixtures for a specific team and league from the Soccerway website. Users can enter the team name and league, and the application provides the last five and next five match fixtures in an organized format. The tool offers a user-friendly interface for easy access and interaction, making it a convenient solution to stay updated with upcoming football fixtures for any chosen team and league.

Nike Web Scraper

The objective of this project is to build a web scraper to get the search results of an item in nike.com. The data was scraped from www.nike.com The features returned are Name, Description, Available In, Price, Old Price Image Link, URL.
