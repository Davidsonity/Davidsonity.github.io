Gender By Voice (Clustering)

The objective of this project is to use several unsupervised machine learning algorithm to group voices. We would be focused on clustering the dataset into male or female. Based upon acoustic properties of the voice and speech. The dataset consists of 3,168 recorded voice samples, collected from male and female speakers.
Some libraries used in this project includes:

  • Pandas
  • Numpy
  • Scikit-learn
  • Seaborn
